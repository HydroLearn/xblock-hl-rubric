A rubric is a tool used to assess student work. Rubrics increase objectivity and reduce subjectivity, and they improve studentsâ€™ ability to meet expectations.

A rubric is made up of a set of [criteria] and each criteria has at least three completion levels with detailed a description of what each criterion would look like completed at the associated level. Lastly, each level should have an assigned point value.

Criteria

A rubric is made up of multiple criteria, usually 3 or more but at times as many as 10-15 for long and involved assignments. Criteria identify the feature or dimension which is to be measured. To determine what your criteria should be:

  1. Identify the list of all criteria, representing the features or dimensions associated with the student work.
  2. Write criteria descriptions and keep them brief and understandable for students to follow.
  3. Put one criterion in each cell in the leftmost column of the rubric.
  4. Add more rows as needed, but reduce the list by removing or combining similar criteria.

Levels

Levels define what different levels at which the assignment meets the expectations set for each criterion. Typically there are 3-5 levels for each criterion. Some dichotomous criteria (e.g., if an answer is right or wrong) might be assigned just 2 levels. To create levels:

  1. Modify the number of levels by adding columns to your rubric.
  2. Then label each level (e.g., low/mid/high, poor/good/excellent, or does not meet expectations/meets expectations/exceeds expectations).
  3. Write a detailed description of what performance would look like at each level. Make sure they are discrete enough to show real differences.

Point Value

After describing what each criterion would look like at each level, assign a point value to each column or cell. Often educators assign a value to each column, but it is also possible to weight criteria differently by assigning different values to each cell.
